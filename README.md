## 2. 使用说明
### 2.1 开发
```
# 克隆项目
git clone https://github.com/LLiuHuan/arco-design-pro-vue.git

# 进入项目目录
cd arco-design-pro-vue

# 安装依赖
yarn install

# 启动服务
yarn dev:mock
```
